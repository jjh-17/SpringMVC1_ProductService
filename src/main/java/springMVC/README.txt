1. 상품 목록
    -ID, 상품명, 가격, 수량
    -Id, 상품명을 눌러 상품 상세로 이동
    -상품 등록 버튼 존재

2. 상품 상세
    -ID, 상품명, 가격, 수량
    -상품 수정/상품 목록으로 이동 가능

3. 상품 등록
    -상품명, 가격, 수량 입력
    -취소 가능

4. 상품 수정
    -상품 ID 수정 불가
    -취소 가능
